### 3. [Data Cleaning Report](./projects/data-cleaning-report/)
**Excel Data Cleaning Project** - Professional data quality assessment and transformation

**Dataset:** Sales transaction data (Date, Client, Contact, Revenue, Profit, Profit Margin)

**Issues Identified & Resolved:**
- Missing values in Revenue column → Replaced with mean value
- Inconsistent spacing in Contact column → Used TRIM() & CLEAN() functions
- Inconsistent text formatting → Used PROPER() function
- Punctuation inconsistencies in Client column → Used Find & Replace method

**Result:** Clean, analysis-ready dataset ✨
