# Sales Dashboard 2025 📊

## Project Overview

Interactive Excel dashboard designed to explore sales performance in a clear and structured way. This dashboard provides comprehensive insights into revenue, product performance, regional metrics, and individual salesperson performance.

✅ **Salesperson Metrics** - Individual performance tracking
✅ **Interactive Visualizations** - Charts and tables for easy decision-making
